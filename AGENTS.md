# 공통 작업 가이드

이 저장소에서 커밋할 때는 아래 규칙을 반드시 지켜주세요.

## 커밋 메시지 규칙

- 커밋 메시지는 **반드시 한국어**로 작성해요.
- 제목(첫 줄)은 **짧고 명료하게** 작성해요.
- 제목은 변경의 핵심만 담고, 불필요한 수식어는 제외해요.
- 제목 아래 한 줄을 비운 뒤, 필요한 경우 상세 내용을 작성해요.
- 상세 내용에는 아래를 포함해요.
  - 무엇을 변경했는지
  - 왜 변경했는지
  - 영향 범위나 주의사항(있다면)

## 권장 형식

```text
[짧고 명료한 한글 제목]

- 변경 사항 1
- 변경 이유
- 영향 범위/주의사항
```

## 예시

```text
로그인 버튼 접근성 개선

- 버튼 텍스트 대비를 WCAG AA 기준에 맞게 조정
- 키보드 포커스 스타일을 명확히 추가
- 로그인 페이지 UI에만 영향
```
