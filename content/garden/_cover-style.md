# Cover Style Guide (Hybrid Guardrail)

이 문서는 `baoyu-cover-image` 스킬로 블로그 커버를 만들 때 톤앤매너를 고정하기 위한 기준입니다.

## 1) 색상 팔레트 (Warm Minimal Editorial)

- Sky/Base: warm family rotation (golden-amber, terracotta-rose, sage-warm, olive-sunset, peach-coral, deep-warm-dusk)
- Ground/Neutral: palette-matched warm neutrals per family
- Portal Contrast: `#FF5D00`, `#2A0E08`
- Soft Neutral Tint: `#E8E0D7`, `#F3EBE2`
- 금지 색상: 형광 계열 네온톤, 순보라 메인톤, 과한 고채도 블루 메인톤

## 2) 구도 (최소 구성)

- 기본 비율: `1200x675` (16:9)
- 중앙에 단일 포털/핵심 오브젝트 1개만 배치
- 보조 도형은 최대 2개까지만 허용
- 넓은 여백 유지 (화면의 60% 이상은 비워두기)
- 깊이감은 2~3 레이어까지만 사용 (과도한 원근 왜곡 금지)
- 같은 구도 연속 사용 금지 (구도 패밀리 로테이션)
  - `single-tree`
  - `double-tree-and-path`
  - `hill-and-river`
  - `greenhouse-grid`
  - `seed-to-sprout`
  - `forest-silhouette`

## 3) 질감/조명

- 부드러운 따뜻한 광원 + 중간 대비
- 종이 질감에 가까운 미세 그레인 허용
- 플랫 벡터/에디토리얼 일러스트 기반, 하드 3D 렌더 느낌 금지
- 과도한 샤픈, 하이 콘트라스트, 과장된 글로우 효과 금지

## 4) 금지 규칙

- 이미지 내부 텍스트 삽입 금지 (`no text`)
- 실사 인물 얼굴/손 클로즈업 금지 (왜곡 리스크)
- 로고 워터마크, 브랜드 마크 무단 삽입 금지
- 과도한 아이콘/오브젝트 난립 금지
- 복잡한 UI 패널/차트/커서 요소 남용 금지

## 6) 고정 파라미터 권장값

- Type: `minimal`
- Palette: `warm`
- Rendering: `flat-vector`
- Text: `none`
- Mood: `subtle`
- Aspect: `16:9`
